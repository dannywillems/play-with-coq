## Play with Coq

```
opam switch create ./ 4.14.2 --no-install
opam install coq -y
```

To use Coq in an interactive environment, use `coqtop`.

To use Coq in VScode: `opam install coq-lsp vscoq-language-server`
Compile using `dune build`.
